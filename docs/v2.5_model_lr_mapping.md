# 模型架構與學習率分組對照表

這份文件詳細說明了 `FusionSpeakerModel` 的各個組成部分，以及它們在程式碼中的位置和對應的學習率設定。

## 1. 整體架構圖

```mermaid
graph TD
    %% 輸入層
    Wav[音訊 Waveform] --> W2V2[Wav2Vec2 Encoder]
    Feat[頻譜 Fbank] --> RES[ResNet Encoder]

    %% 特徵提取
    W2V2 --> W_Emb[Wav2Vec2 Embedding]
    RES --> R_Emb[ResNet Embedding]

    %% 融合層 (Fusion)
    W_Emb & R_Emb --> Concat[串接特徵]
    Concat --> LN[LayerNorm (ln)] 
    
    %% 分支 1: 語者識別 (ArcFace)
    LN --> PROJ[ArcMarginProduct (projection)]
    PROJ --> SpkID[語者分類 ID]

    %% 分支 2: 語言對抗 (Language Adversarial)
    LN --> GRL[Gradient Reversal Layer]
    GRL --> L_HEAD[Language MLP (language_head)]
    L_HEAD --> LangID[語言分類 ID]

    %% 標示對應 LR Group
    style W2V2 fill:#e3f2fd,stroke:#1565c0
    style RES fill:#fff3e0,stroke:#ef6c00
    style LN fill:#f3e5f5,stroke:#7b1fa2
    style L_HEAD fill:#fce4ec,stroke:#c2185b
    style PROJ fill:#e8f5e9,stroke:#2e7d32
```

## 2. 詳細對照表

### A. Wav2Vec2 分支 (藍色)
負責處理原始音訊波形。

| 組件名稱 | 程式碼變數 (`fusion_model.py`) | LR Group (`yaml`) | 說明 |
| :--- | :--- | :--- | :--- |
| **Backbone** | `self.w2v2.backbone` | `w2v2_backbone` | Wav2Vec2 的 Transformer 主幹層 (大量的預訓練參數) |
| **Head** | `self.w2v2.pool`, `self.w2v2.linear` | `w2v2_head` | Wav2Vec2 的池化層與線性輸出層 |

### B. ResNet 分支 (橘色)
負責處理 Fbank 頻譜特徵。

| 組件名稱 | 程式碼變數 (`fusion_model.py`) | LR Group (`yaml`) | 說明 |
| :--- | :--- | :--- | :--- |
| **Backbone** | `self.resnet.front` | `resnet_backbone` | ResNet 的卷積層 (Conv Layers) |
| **Head** | `self.resnet.pooling`, `self.resnet.bottleneck` | `resnet_head` | ResNet 的 ASP 池化與 Bottleneck 層 |

### C. 融合層 (紫色) 🟣
這是您問到的 `fusion` 部分。

| 組件名稱 | 程式碼變數 (`fusion_model.py`) | LR Group (`yaml`) | 說明 |
| :--- | :--- | :--- | :--- |
| **LayerNorm** | `self.ln` | `fusion` | **這就是 `fusion`！**<br>程式碼: `self.ln = nn.LayerNorm(embed_dim)`<br>作用: 將串接後的特徵做規範化，讓後面的分類器更好訓練。 |

### D. 語言對抗分支 (粉紅色) 🔴
這是您問到的 `language_head` 部分。

| 組件名稱 | 程式碼變數 (`fusion_model.py`) | LR Group (`yaml`) | 說明 |
| :--- | :--- | :--- | :--- |
| **Language MLP** | `self.language_head` | `language_head` | **這就是 `language_head`！**<br>程式碼:<br>`self.language_head = nn.Sequential(`<br>`  nn.Linear(embed_dim, 256),`<br>`  nn.ReLU(),`<br>`  nn.Linear(256, num_languages)`<br>`)`<br>作用: 一個小型的神經網路，用來判斷現在是哪種語言。 |

### E. 語者識別分支 (綠色)
最後輸出的分類層。

| 組件名稱 | 程式碼變數 (`train.py`) | LR Group (`yaml`) | 說明 |
| :--- | :--- | :--- | :--- |
| **ArcFace** | `projection` | `projection` | ArcMarginProduct 分類層，負責算出 3000+ 位語者的分數。 |

## 3. 設定建議 (符合您的需求)

若您希望簡化管理，只需關注以下四個大區塊：

1. **Wav2Vec2**: 設定 `w2v2_backbone` 和 `w2v2_head` 為相同值 (例如 0.1)
2. **ResNet**: 設定 `resnet_backbone` 和 `resnet_head` 為相同值 (例如 0.2)
3. **去語言 (Language)**: 設定 `fusion` 和 `language_head` 為相同值 (例如 1.0)
    * `fusion` 是特徵融合點，`language_head` 是特徵後的分類器，兩者都是「語言對抗」這件事的關鍵新層。
4. **ArcFace**: 設定 `projection` (例如 1.0)

這樣設定就非常清晰了！
