# Implement Fusion Bottleneck Layer

## Goal
Add a bottleneck layer to `FusionSpeakerModel` to compress the concatenated features (from Wav2Vec2 and ResNet) from 512 dimensions to 256 dimensions. This aligns with the standard practice of using a compact embedding space for speaker verification.

## Architecture Change

**Current:**
```
[W2V2 (256)] + [ResNet (256)] -> [Concat (512)] -> [LayerNorm] -> [Output (512)]
```

**Proposed:**
```
[W2V2 (256)] + [ResNet (256)] -> [Concat (512)] -> [LayerNorm] -> [Bottleneck (Linear 512->256 + BN)] -> [Output (256)]
```

## Proposed Changes

### Model Code

#### [MODIFY] [fusion_model.py](file:///home/brant/Project/tidyVoice/wespeaker/wespeaker/models/fusion_model.py)

1.  **__init__**:
    *   Add `fusion_dim` calculation (sum of w2v2 and resnet dims).
    *   Add `self.bottleneck = nn.Sequential(nn.Linear(fusion_dim, embed_dim), nn.BatchNorm1d(embed_dim))`.
    *   Update `self.language_head` to take `embed_dim` (256) instead of `fusion_dim` (512).
2.  **forward**:
    *   Pass `combined` through `self.bottleneck` after `self.ln`.
