## Phase 3: Focal Loss + 解凍 Whisper 最後 1 層 (⏳ 下一步)

> [!IMPORTANT]
> Phase 2 已確認瓶頸在 Whisper 特徵層面。Phase 3 採用雙管齊下策略：改造 Loss 函數 + 解凍 Encoder 最後一層。

### 架構圖
```
Whisper Encoder [..., Layer 31 (🔒), Layer 32 (🔥)] → model_seq Conv1d (🔥) → emotion_layer (🔥)
                                     ↑ 解凍！~16.8M     ↑ 維持 526K          ↑ 維持 66K
```

### 改動 1：Focal Loss (取代 CrossEntropyLoss)

**為什麼要換？**
- 目前的 `CrossEntropyLoss(weight=...)` 只是給 Fear「更重的分數」，但 Neutral/Happy 佔 96% 的樣本量，它們產生的大量「已經猜對了但還是有梯度」的訊號會淹沒 Fear 的學習信號
- **Focal Loss** 的數學特性：`FL(p) = -α(1-p)^γ * log(p)`
  - 當 `p` 很高（模型已經猜對）→ `(1-p)^γ` 趨近 0 → 梯度幾乎歸零
  - 當 `p` 很低（模型猜錯，常見於 Fear）→ `(1-p)^γ` 接近 1 → 梯度完整保留
  - 效果：**GPU 每一步的算力幾乎全部集中在猜錯的困難樣本上**

**超參數**：
- `gamma = 2.0`（標準值，對簡單樣本的抑制力道適中）
- `alpha`：沿用現有的 sqrt 逆頻率 class weights

**程式碼改動** ([train_4class.py](file:///home/brant/Project/SAILER_test/train_4class.py))：
```python
class FocalLoss(nn.Module):
    def __init__(self, weight=None, gamma=2.0):
        super().__init__()
        self.weight = weight
        self.gamma = gamma

    def forward(self, inputs, targets):
        ce_loss = F.cross_entropy(inputs, targets, weight=self.weight, reduction='none')
        pt = torch.exp(-ce_loss)  # 模型對正確類別的信心
        focal_loss = ((1 - pt) ** self.gamma) * ce_loss
        return focal_loss.mean()

# 替換原本的 criterion
criterion = FocalLoss(weight=class_weights, gamma=2.0)
```

### 改動 2：解凍 Whisper Encoder 最後 1 層

**為什麼只解凍 1 層？**
- Whisper-large-v3 有 32 層 Encoder，每層 ~16.8M 參數
- 解凍 1 層 → 可訓練參數從 526K 增加到 **~17.3M**（仍只佔總量的 1.1%）
- 最後一層產出的特徵直接餵進 `model_seq`，讓它學會強化恐懼的聲學訊號（語音發抖、音調不穩）

**程式碼改動** ([train_4class.py](file:///home/brant/Project/SAILER_test/train_4class.py))：
```python
# Phase 3: 解凍 Whisper Encoder 最後 1 層
last_layer = model.whisper.encoder.layers[-1]
for param in last_layer.parameters():
    param.requires_grad = True
```

### 三層 Discriminative LR
```python
optimizer = torch.optim.AdamW([
    {'params': model.whisper.encoder.layers[-1].parameters(), 'lr': 1e-5},  # Whisper 最後層：極慢
    {'params': model.model_seq.parameters(), 'lr': 1e-4},                   # Conv1d：小步微調
    {'params': model.emotion_layer.parameters(), 'lr': 1e-3},               # Head：正常學習
], weight_decay=0.01)
```

### 訓練策略
- **Epochs**: 30 (搭配 patience=8 Early Stop)
- **Batch Size**: 維持 64
- **Warmup**: 前 2 Epoch 凍結 Whisper 最後層，只跑 Focal Loss 讓 Head 先適應 → 第 3 Epoch 開始解凍

> [!WARNING]
> 若 Epoch 3 開始 Val Loss 暴漲，應立即降低 Whisper 層 LR 至 `5e-6`。
