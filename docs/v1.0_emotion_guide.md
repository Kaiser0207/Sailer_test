# IndexTTS2 情緒控制完全指南 (Emotion Control Guide)

IndexTTS2 的核心強項在於其強大的「零樣本情緒控制」能力。它允許將**說話者的音色 (Timbre)** 與 **說話的情緒 (Emotion)** 完全解耦。

這意味著：你可以讓「A 的聲音」用「B 的情緒」來說話，或者直接指定「我要 50% 快樂 + 50% 驚訝」的聲音。

本指南將詳細說明 4 種情緒控制模式及其操作方式。

---

## 1. 模式一：參考音訊情緒 (Emotion from Reference Audio)

這是最直覺的模式。你提供一段「帶有特定情緒的錄音」（不需要是同一個人的聲音），模型會模仿這段錄音的情緒語調，但保持原本設定的說話人音色。

### 原理
- **Speaker Prompt (`spk_audio_prompt`)**：決定「是誰在說話」（音色）。
- **Emotion Prompt (`emo_audio_prompt`)**：決定「用什麼語氣說話」（情緒）。

### 如何使用 (Python)

在 `infer_v2.py` 中：

```python
tts.infer(
    spk_audio_prompt="my_voice.wav",       # 決定音色 (我是誰)
    text="這真是一個天大的好消息！",          # 說什麼內容
    output_path="output_ref.wav",
    
    # --- 情緒設定 ---
    emo_audio_prompt="laughing.wav",       # 決定情緒 (例如一段大笑的錄音)
    emo_alpha=1.0                          # 情緒強度 (0.0 ~ 1.0)
)
```

**參數說明：**
*   `emo_alpha`：數值越大，模仿的情緒越強烈。預設為 1.0。

---

## 2. 模式二：情緒向量控制 (Emotion Vector Control)

如果你沒有參考錄音，或者想要精細調配情緒（例如：三分怒七分悲），可以使用情緒向量。IndexTTS2 支援 8 種基礎情緒維度。

### 支援的情緒維度 (順序固定)
1.  **Happy (高興)**
2.  **Angry (憤怒)**
3.  **Sad (悲傷)**
4.  **Afraid (恐懼)**
5.  **Disgusted (反感)**
6.  **Melancholic (低落)**
7.  **Surprised (驚訝)**
8.  **Calm (平靜/自然)**

### 如何使用 (Python)

```python
# 設定情緒向量：[喜, 怒, 哀, 懼, 厭, 鬱, 驚, 平]
# 範例：非常驚訝 (0.8) 帶一點高興 (0.2)
my_emotion = [0.2, 0, 0, 0, 0, 0, 0.8, 0]

tts.infer(
    spk_audio_prompt="my_voice.wav",
    text="哇！這禮物太棒了吧！",
    output_path="output_vec.wav",
    
    # --- 情緒設定 ---
    emo_vector=my_emotion,                 # 傳入向量
    emo_alpha=0.8                          # 整體強度建議設為 0.6 ~ 0.9 效果較自然
)
```

**技巧：**
*   陣列中的數值通常在 0.0 到 1.0 之間。
*   混合使用時，總和不需要等於 1，模型會自動正規化。

## 3. 模式三：文字描述情緒 (Emotion from Text Description)

這是最先進的模式。你不需要調參數，直接用文字描述你想要的情緒，模型會自動理解並轉換成對應的情緒向量。

### 原理
模型內部使用了一個微調過的 Qwen (通義千問) 大語言模型來理解你的情緒描述。

### 如何使用 (Python)

有兩種用法：

#### 用法 A：明確指定情緒描述 (`emo_text`)
將「台詞」與「情緒指導」分開。

```python
tts.infer(
    spk_audio_prompt="my_voice.wav",
    text="你怎麼可以這樣對我？",             # 台詞
    output_path="output_text_emo.wav",
    
    # --- 情緒設定 ---
    use_emo_text=True,                     # 啟用文字情緒模式
    emo_text="絕望地哭喊，聲音顫抖，充滿了被背叛的痛苦",  # 情緒指導
    emo_alpha=0.7
)
```

#### 用法 B：從台詞自動推論
不指定 `emo_text`，模型會自己讀台詞來猜測該用什麼語氣。

```python
tts.infer(
    spk_audio_prompt="my_voice.wav",
    text="我真的好想你，你什麼時候才回來...",  # 台詞 (模型會自動分析出悲傷/低落)
    output_path="output_auto_emo.wav",
    
    # --- 情緒設定 ---
    use_emo_text=True,
    emo_text=None,                         # 留空，讓模型分析 text
    emo_alpha=0.6
)
```

---

## 4. 模式四：隨機情緒 (Random / Stochastic)

如果你想要生成多樣化、不確定性的結果，可以啟用隨機模式。這在尋找靈感時很有用。

### 如何使用 (Python)

```python
tts.infer(
    spk_audio_prompt="my_voice.wav",
    text="這是一個隨機測試。",
    output_path="output_random.wav",
    
    # --- 情緒設定 ---
    use_random=True,                       # 啟用隨機性
    # 也可以搭配 emo_vector 使用，會在指定情緒的基礎上增加隨機變化
    emo_vector=[0.5, 0, 0, 0, 0, 0, 0, 0] 
)
```

---

## 總結：我該用哪種？

| 需求 | 推薦模式 | 說明 |
| :--- | :--- | :--- |
| **我有參考素材** | **模式一 (Reference Audio)** | 最自然，直接模仿錄音的語氣。 |
| **我要精確控制** | **模式二 (Vector)** | 適合工程化調優，例如製作遊戲語音時固定參數。 |
| **我只想要方便** | **模式三 (Text Description)** | 最簡單，像寫劇本提示一樣 (Prompting)。 |
| **我想抽卡/探索** | **模式四 (Random)** | 產生意想不到的語氣變化。 |

## 參數調整建議

- **`emo_alpha`**：
  - **模式一**：建議 `1.0`。
  - **模式二/三**：建議 `0.6` ~ `0.9`。如果設太高，聲音可能會變得過度誇張或出現雜訊。
