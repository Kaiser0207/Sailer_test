## 2️⃣ V3.5: 官方 Whisper + RoBERTa 融合模型 (Run 1)
第一次融合實驗。我們把官方 Whisper 與 RoBERTa 的權重雙雙「凍結」，只訓練 `SAILER_Model` 的特徵融合頭與分類器。

- **實驗 ID**：`20260412_173826_SAILER_BestSingleSystem_v3.5_OfficialFusion`
- **關鍵超參數**：Batch Size `32`, LR `0.0005`, AVD Weight `8.0`
- **驗證集最佳表現 (Epoch 3)**：
  - **Macro F1**：`0.3872` (接近 `0.39`)
  - **Accuracy**：`0.478` (大約 `0.48`)

## 3️⃣ V3.5: 降速深潛求精模型 (Run 2 - 最新)
為了解決 Run 1 過早收斂的問題，我們全方位修改了超參數，讓訓練步伐平穩扎實。

- **實驗 ID**：`20260413_125641_SAILER_BestSingleSystem_v3.5_OfficialFusion`
- **關鍵超參數**：Batch Size `64`, LR `0.0003`, Weight Decay `0.0001`, Warmup `0.05`
- **驗證集表現 (最佳 Epoch)**：
  - **Macro F1**：`0.35`
  - **Accuracy**：`0.4186` (約 41.8%)

## 💡 下一步會議討論重點
1. **是否推上 Test2 進行決戰？**
   目前的 V3.5 (Run 1) 在 Validation Set 上有強大的 `0.39` F1 表現（少數類別甚至能穩定召回）。強烈建議抽出兩次 V3.5 的最佳 checkpoint 正式對 Test2 跑一次評估腳本，看能否一舉打破 Baseline (0.28) 的斷層魔咒。
2. **要不要解凍 (Unfreeze) 機制？**
   目前 V3.5 取得的所有成果，都是建立在主幹網路（Whisper / RoBERTa）**完全凍結**的狀態下。若硬體極限允許，我們下一週也可嘗試 "Fine-tuning top 2 layers" 的解凍策略，讓預訓練權重針對目前的聯合機制做出終極妥協，引爆下一波 SOTA。
